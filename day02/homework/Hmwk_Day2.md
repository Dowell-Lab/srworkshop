## Day 2 Homework
### Objectives
- Make a new directory and figure out how to copy another directory containing multiple files into your new directory.
- Practice Vim.
- Watch day 3 videos.

### Instructions
1. First, log into the super computer and practice `mkdir` by making a directory called `<your_name>_Rocks` (so if my name is Laura it'd be Laura_Rocks).

2. Use `rsync` to copy the following directory into the `<your_name>_Rocks` directory: `/scratch/Shares/public/sread/data_files/day2/bedfiles` 
	- Hint: This requires the recursive flag (you can use `man rsync` or `rsync --help` to figure out this flag)

3. **THIS IS MOST IMPORTANT:** If you haven't yet, work through the Vim tutorial available on the server (`vimtutor`).

4. Watch [day 3 videos](https://github.com/Dowell-Lab/srworkshop/blob/main/day03/README.md).
