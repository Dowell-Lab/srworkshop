There are three scripts in this directory. To use this directory you will need to:
1. Change the in and out direcory in runsbatchRNAscript.sh.
2. Change the slurm header in sbatchRNAscript.sh.



runsbatchRNAscript.sh will run sbatchRNAscript.sh one time for each bam file.
sbatchRNAscript.sh will run RNAseq_featurecounts.R.
